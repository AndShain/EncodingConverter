Для работы программы нужен файл с настройками - Settings.xml

Список настроек:
directoryPath - путь до папки с файлами, которым нужно поменять кодировку
backupDirectoryPath - папка, куда будут сохранены файлы с новой кодировкой(если directoryPath = backupDirectoryPath исходные файлы будут перезаписаны)
sourceEncoding - кодировка исходных файлов
destinationEncoding - кодировка, в которую будут преобразованы исходные файлы
extensions - расширения файлов, которым нужно сменить кодировку

P.S. В данный момент приложение не умеет определять кодировку исходного файла автоматически, поэтому при указании неправильной исходной кодировки, выходные файлы будут повреждены(для теста программы нужно указать backupDirectoryPath отличную от directoryPath)
