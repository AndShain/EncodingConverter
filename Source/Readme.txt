Для работы программы нужен файл с настройками - Settings.xml (файл должен находиться в папке с исполняемым файлом)
Программа записывает логи в файл Logs.txt(создается в папке с исполняемым файлом)

Список настроек:
directoryPath - путь до папки с файлами, которым нужно поменять кодировку
destinationEncoding - кодировка, в которую будут преобразованы исходные файлы
extensions - расширения файлов, которым нужно сменить кодировку

Значения, которые может принимать destinationEncoding в settings.xml:
utf8
1251

Если вы хотите добавить другие кодировки, отредактируйте метод ConverTextToEncoding(string text) в Converter.cs
